项目统计了自2023年8月25日以来B站每日的热门视频数据，数据来源于B站的API接口，每日更新一次。

由于程序没有设置Cookie，数据不存在个性化（冷知识：每个人的B站热门都不一样）。

注：
1. 每日的数据截至程序运行
2. 数据仅供学习参考
3. 如需运行项目中的程序，切记不要过度频繁使用！！！否则造成的后果与本人无关
